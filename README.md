# 🚀 Axiom Taste Refiner

An open-source prompt styling injector framework designed to eliminate generic, robotic LLM outputs. Infuse premium brand architecture, raw technical tone filters, and customized execution rules into your AI prompts instantly—100% locally.

## 🔥 Key Benefits
- **Zero Hallucination Overlays:** Works purely through architectural reinforcement modifiers.
- **Dynamic Taste Matrices:** Hot-swap between Premium Luxury, Industrial Cyberpunk, or Offensive Hacker profiles.
- **Client-Side Sovereignty:** Execution runs completely in local runtime memory. Zero data leaves the machine.

## 📦 Integration

```javascript
import { refinePrompt } from './refiner.js';

// Refine any generic prompt into an architectural masterpiece
const raw = "Write a sales message for a software utility.";
const premiumPrompt = refinePrompt(raw, 'premium');

console.log(premiumPrompt);
```
